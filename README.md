    ## Project Overview
    Analyzed Zomato restaurant dataset with 9.5K+ restaurants to understand location, cuisine, and delivery trends.
    
    ## Key Insights
    - New Delhi has max restaurants & 35.4% delivery penetration
    - Only 25.6% restaurants offer online delivery
    - Avg rating is 2.67 vs target 4.0
    - North Indian is the most popular cuisine
    
    ## Tools Used
    Power BI, DAX, Data Cleaning
