# Zomato Restaurant Analysis - Power BI

## Project Overview
Analyzed Zomato restaurant dataset with 9.5K+ restaurants to understand location, cuisine, and delivery trends across India.

## Dashboard
![Zomato Dashboard](zomato_dashboard.png)

## Key Insights
- **New Delhi** has max restaurants & 35.4% delivery penetration
- **Only 25.6%** restaurants offer online delivery 
- **Avg rating is 2.67** vs target 4.0
- **North Indian** is the most popular cuisine
- **Tort Bangalore** is the highest rated restaurant - 4.9

## Tools Used
- Power BI for visualization
- DAX for measures
- Excel/Data Cleaning for preprocessing

## Features
- City, Online Delivery, Table Booking filters
- KPI cards for Total Restaurants & Avg Rating
- Charts for Top 10 Cities, Rating Distribution

## How to Use
Download the `.pbix` file from this repo and open in Power BI Desktop.
